# 📚 Library Management System

## 📌 Project Description

The Library Management System is a simple console-based Java application that helps manage books in a library. It allows users to add books, view available books, issue books, and return them.

This project demonstrates basic concepts of Java programming in a practical and easy-to-understand way.

---

## 🎯 Objectives

* To understand the use of classes and objects in Java
* To implement real-world problem solving using programming
* To learn how to use ArrayList for dynamic data storage
* To perform basic operations like add, display, update, and delete

---

## 🛠️ Technologies Used

* Java 
* OOP Concepts (Class, Object)
* ArrayList
* Scanner (for user input)

---

## ✨ Features

* ➕ Add new books
* 📖 View all books
* 📕 Issue a book
* 📗 Return a book
* ❌ Exit the system

---

## 🧠 Working of the Project

1. The program displays a menu with different options.
2. The user selects an option by entering a number.
3. Based on the choice:

   * Books can be added to the system
   * All books can be displayed
   * A book can be issued (marked as unavailable)
   * A book can be returned (marked as available)
4. The program continues until the user selects the exit option.

---

## 🧱 Project Structure

* **Book Class**

  * Stores book details such as ID, title, author, and status
  * Contains a method to display book information

* **Main Class (LibraryManagementSystem)**

  * Contains the main method
  * Handles user input and menu operations
  * Manages the list of books using ArrayList

---

## ▶️ How to Run the Project

1. Open any Java IDE (VS Code / Eclipse / IntelliJ)
2. Save the file as:

   ```
   LibraryManagementSystem.java
   ```
3. Compile the program:

   ```
   javac LibraryManagementSystem.java
   ```
4. Run the program:

   ```
   java LibraryManagementSystem
   ```
